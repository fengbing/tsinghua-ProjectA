## 名称

飞机躲避障碍物核心玩法

## 概要

玩家控制一架持续向前飞行的飞机，在由立方体障碍物组成的 3D 空间中穿梭躲避，尽可能长时间地存活或飞行更远距离。游戏强调拟真的飞行手感（重量感、推力感和惯性），而非简单的平移移动。

## 详细需求

- 飞机默认沿一个主方向（例如世界 Z 轴正方向）持续前进，可配置基础巡航速度。
- 玩家可以通过键盘 `W/A/S/D` 对飞机进行控制：
  - `W/S`：控制俯仰（机头上仰 / 下俯），影响垂直方向和角度。
  - `A/D`：主要控制横滚或偏航，用于横向躲避障碍。
  - 至少一个方向键（或组合）用于影响推力（“魔法力”），从而改变飞机当前速度或上升能力。
- 飞机应具有明显的物理特性：
  - 受重力影响，若不输入控制，飞机会逐渐下坠或偏离安全通道。
  - 具有惯性：松开控制键后不会立刻停止，而是逐渐变化。
  - 可配置最大速度、旋转速度和加速度，以调节手感。
- 游戏应提供清晰的失败条件：
  - 飞机与任意障碍物（由 Cube 组成）发生碰撞视为失败。
  - 失败时应有明显反馈（如摄像机震动、粒子特效、声音或 UI 提示）。
- 游戏至少提供一种简单的胜利 / 评分机制，例如：
  - 以存活时间、飞行距离或通过的障碍段数作为得分。
  - 得分在 UI 上实时展示。

## 验收标准

- 在 Unity 编辑器中按下 Play 后，飞机自动开始沿预定方向飞行。
- 通过 `W/A/S/D` 可以明显改变飞机飞行方向和姿态，且感受到重力与惯性。
- 飞机与可见障碍物碰撞时，游戏会进入 Game Over 状态，并给出反馈。
- 屏幕上实时显示至少一个与进度相关的数值（时间 / 距离 / 得分）。
